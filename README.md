# Denex
> **A script that scrapes information about an anime or manga from myanimelist.net**

## Usage

```bash
$ denex
```

![first](https://i.imgur.com/QLVUWEt.png)

```bash
$ denex
Enter the name: 
Anime or manga[A/m]: 
```
![second](https://i.imgur.com/3KokUpK.png)

## Dependencies
- sed
- jq
- curl
- awk
